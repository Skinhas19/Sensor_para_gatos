# Sensor_para_gatos
# 🐱 Detector de Movimento com ESP32 e Telegram

Este projeto utiliza um **ESP32** e um **sensor de movimento (PIR)** para detectar quando há movimento (por exemplo, seu gato pedindo comida 😺).  
Quando o movimento é identificado, o ESP32 envia uma **notificação automática pelo Telegram** utilizando a API oficial do Telegram Bot.
